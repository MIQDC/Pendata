Apa Itu Outlier Deteksi?#

Outlier adalah data yang berbeda secara signifikan dari data lain dalam dataset. Outlier dapat disebabkan oleh kesalahan pengukuran, kesalahan input data, atau memang fenomena khusus yang unik dibandingkan dengan data lainnya. Kehadiran outlier dapat mempengaruhi analisis data dan model machine learning. Sedangkan deteksi outlier adalah metode untuk mengidentifikasi anggota dalam kelompok yang menyimpang jauh dari norma.

Kode diatas digunakan untuk menginstal dua pustaka Python, yaitu pymysql dan psycopg2, yang berfungsi sebagai konektor untuk basis data (database). def get_mysql_data():
    conn = pymysql.connect(
        host="mysql-207e66db-iqbal.h.aivencloud.com",

        password="AVNS_moPYLh4RrOq11aEBY_x",

    cursor = conn.cursor()
    cursor.execute("SELECT * FROM iris_dataset")

    columns = [desc[0] for desc in cursor.description]  # Ambil nama kolom
    cursor.close()
    conn.close()

    return pd.DataFrame(data, columns=columns)

# Ambil data dari kedua database
df_postgresql = get_pg_data()
df_mysql = get_mysql_data()

# Gabungkan berdasarkan kolom 'id' dan 'Class'
df_merged = pd.merge(df_mysql, df_postgresql, on=["id", "class"], how="inner")

# Ambil data fitur numerik
feature_columns = ["petal_length", "petal_width", "sepal_length", "sepal_width"]
data_values = df_merged[feature_columns].values

# Cetak semua data hasil gabungan tanpa indeks
print(df_merged.to_string(index=False))

# Ambil referensi dari baris terakhir (baris ke-152 jika dihitung dari 1, atau index -1)
reference_point = data_values[-1]

def compute_distances(data, reference):
    return np.array([euclidean(row, reference) for row in data])

# Hitung jarak Euclidean dari setiap baris ke referensi
df_merged["distance"] = compute_distances(data_values, reference_point)

# Tentukan threshold outlier berdasarkan persentil ke-93.5 (bisa disesuaikan)
threshold = np.percentile(df_merged["distance"], 93.5)
df_merged["outlier"] = df_merged["distance"] > threshold

# Cetak hasil data dengan outlier
print(df_merged.to_string(index=False))

# Visualisasi scatter plot dengan warna berdasarkan outlier
plt.figure(figsize=(8, 6))
sns.scatterplot(
    x=df_merged["sepal_length"], y=df_merged["sepal_width"],
    hue=df_merged["outlier"], palette={False: "blue", True: "red"}
)
plt.title("Outlier Detection (Sepal)")
plt.show()

plt.figure(figsize=(8, 6))
sns.scatterplot(
    x=df_merged["petal_length"], y=df_merged["petal_width"],
    hue=df_merged["outlier"], palette={False: "blue", True: "red"}
)
plt.title("Outlier Detection (Petal)")
plt.show()

Kode idiatas bertujuan untuk mengambil data dari dua database yang berbeda, yaitu PostgreSQL dan MySQL, kemudian menggabungkannya, menghitung jarak Euclidean untuk mendeteksi outlier, serta memvisualisasikan hasilnya.

Pertama, kode menggunakan pustaka psycopg2 untuk mengambil data dari PostgreSQL melalui fungsi get_pg_data(). Data ini diambil dari tabel iris_postgresql. Kemudian, pustaka pymysql digunakan dalam fungsi get_mysql_data() untuk mengambil data dari MySQL dari tabel irismysql. Kedua dataset ini dikonversi menjadi DataFrame Pandas.

Setelah data diambil, kedua DataFrame digabungkan berdasarkan kolom id dan class dengan metode inner join, sehingga hanya baris dengan nilai id dan class yang cocok di kedua tabel yang disertakan dalam dataset akhir (df_merged). Selanjutnya, hanya kolom fitur numerik yaitu petal_length, petal_width, sepal_length, dan sepal_width yang diekstrak sebagai array NumPy (data_values).

Jarak Euclidean dihitung antara setiap baris data dengan baris terakhir sebagai referensi, menggunakan fungsi compute_distances(), yang memanfaatkan scipy.spatial.distance.euclidean. Hasil perhitungan ini disimpan dalam kolom baru distance pada df_merged. Untuk mendeteksi outlier, ditentukan threshold berdasarkan persentil 93.5%, artinya jika nilai jarak lebih dari threshold ini, data tersebut dianggap outlier. Data yang termasuk outlier ditandai dengan True dalam kolom outlier.

Akhirnya, kode mencetak seluruh dataset hasil pengolahan, kemudian menggunakan Seaborn dan Matplotlib untuk membuat scatter plot. Dua grafik dibuat: satu memvisualisasikan outlier berdasarkan sepal_length vs sepal_width, dan yang lainnya berdasarkan petal_length vs petal_width. Data yang bukan outlier ditampilkan dalam warna biru, sementara outlier diberi warna merah.

# Hitung total outlier
total_outliers = df_merged["outlier"].sum()
print(f"Total Outlier: {total_outliers}")

Kode tersebut menghitung jumlah total outlier dalam suatu DataFrame bernama df_merged. Pada baris pertama, df_merged["outlier"].sum() digunakan untuk menjumlahkan nilai dalam kolom "outlier", yang diasumsikan berisi nilai biner (misalnya, 1 untuk outlier dan 0 untuk non-outlier). Hasil dari operasi tersebut kemudian disimpan dalam variabel total_outliers. Pada baris berikutnya, hasil perhitungan ini ditampilkan ke layar menggunakan print(f"Total Outlier: {total_outliers}"), yang memanfaatkan f-string untuk menyisipkan nilai total_outliers langsung ke dalam string output.

# Urutkan data berdasarkan jarak dari terkecil ke terbesar
df_sorted = df_merged.sort_values(by="distance", ascending=True)

# Tampilkan hasilnya
print(df_sorted.to_string(index=False))

Kode tersebut melakukan pengurutan data dalam sebuah DataFrame bernama df_merged berdasarkan nilai dalam kolom "distance". Proses ini dilakukan menggunakan fungsi .sort_values(), dengan parameter by="distance" yang menunjukkan bahwa pengurutan dilakukan berdasarkan kolom "distance", serta ascending=True yang menandakan bahwa pengurutan dilakukan dari nilai terkecil ke terbesar. Hasil dari proses pengurutan ini kemudian disimpan dalam variabel baru bernama df_sorted. Setelah pengurutan selesai, hasilnya ditampilkan ke layar dalam bentuk teks menggunakan print(df_sorted.to_string(index=False)), yang memastikan bahwa seluruh isi DataFrame dicetak tanpa menyertakan indeks.
